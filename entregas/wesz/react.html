O que é React?

O React é uma das mais populares bibliotecas do JavaScript. Ela possui código aberto e suas funcionalidades estão voltadas para a construção de uma interface gráfica que torne a experiência do usuário mais eficiente.

Além disso, essa biblioteca pode ser vista como um Framework, possui suporte à reusabilidade de componentes e é capaz de melhorar o SEO de uma página, pois é capaz de melhorar o processamento da aplicação.
