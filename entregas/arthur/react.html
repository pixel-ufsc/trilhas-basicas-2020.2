O que é React?

React é uma biblioteca de Javascript open-source criada pelo Facebook. O React é utilizado para criar interfaces de usuário, ou seja, no front-end. Dessa forma, ele é a parte do view de uma aplicação MVC (model, view e controller). Um dos aspectos mais interessantes do React é que é possível criar elementos de HTML customizáveis que podem ser reutilizados, esses elementos são chamados de Componentes
