## Site da Pixel

Projeto criado pelo [Higor](https://github.com/higorpo), disponibilizado no [figma](https://www.figma.com/file/Gf6FdekAmKTSuHztgJDYmR/Trilhas-b%C3%A1sicas---Site-da-Pixel?node-id=0%3A1), para o desenvolvimento na Trilha Básica dos trainees de 20.2.

O objetivo é a aquisição de boas práticas escrita código, além do aprendizado e comparação de scripts entre membros.
