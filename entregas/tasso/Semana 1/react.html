O que é React?

React é uma biblioteca de JavaScript. Para que a interface de usuário (UI) interativa seja cada vez mais fácil! A função do React dentro da aplicação é atualizar e renderizar os componentes de forma eficaz, fazendo isso apenas quando os dados mudam. É mantido pelo Facebook, Instagram, outras empresas e uma comunidade de desenvolvedores individuais. E utilizado nos sites da Netflix, Imgur, Feedly, Airbnb, SeatGeek, HelloSign, Walmart, além, é claro, de muito outros. Um canal muito bom para acompanhar que sempre lança novas informações sobre o tema é a Rocketseat!
