O que é React?

React é uma das bibliotecas mais populares da linguagem Javascript. Essa biblioteca é utilizada para criar interfaces de interação com o usuário. Foi desenvolida pelo Facebook em 2011 e publicada em 2013 como uma biblioteca de código aberto, o que ajudou na sua expansão. Atualmente é usada em larga escala por grandes companias, como a Netflix. Além disso, o React se carateriza por ser de fácil aprendizagem e utilização.
