Metodologias ágeis - Apresentando o Scrum

Scrum é uma metodologia ágil para gestão e planejamento de projetos de software, no scrum, os projetos são divididos em ciclos que geralmente são mensais, os Spints, o Sprint representa um Time Box dentro do qual um conjunto de atividades deve ser executado.
Metodologias ágeis de desenvolvimento de software são iterativas, ou seja, o trabalho é dividido em iterações, que são chamadas de Sprints no caso do Scrum.
