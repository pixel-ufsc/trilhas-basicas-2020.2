O que é React?

O React é uma biblioteca Javascript que tem como objetivo tornar a criação de interface de usuários facilitada. Tem como principal característica ser declarativo, o que torna o código envolvido mais simples e menor. Por ser uma biblioteca e não um framework, o react é também conhecido por sua flexibilidade e menor complexidade. O react trabalha com componentes: isso significa que a página tem sua estrutura formada por pequenas partes. Esse processo torna porções da página reutilizáveis, o que facilita o trabalho do programador. O react também tem uma solução para aplicações móveis: o React Native.
