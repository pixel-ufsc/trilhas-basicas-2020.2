O QUE É REACT?

Significado e História

Definido por seus criadores, React é "uma bibilioteca JavaScript declarativa, eficiente e flexível para a criação de interfaces de usuário (UI)". Essa biblioteca surgiu em 2011, no Facebook, e até hoje é usada na interface do mural de notícias da rede social. Em 2012, também passou a fazer parte da tecnologia do Instagram. E em 2013, seu código foi aberto, gerando grande popularização.

O que é?

É uma biblioteca

Isso significa que React não é um framework, mas sim uma coleção de funcionalidades que podem ser "chamadas" pelo desenvolvedor para resolver algum problema. Bibliotecas são mais flexíveis e menos complexas que frameworks. No caso do React, sua única e principal função é a criação de interfaces de usuário, que organiza o que é mostrado para o usuário final na tela.

É Javascript

Grande referência para quem trabalha com desenvolvimento front-end, Javascript é a linguagem mais utilizada pelos desenvolvedores atualmente. Esse crescimento está diretamente relacionado à “revolução” trazida por ferramentas como NodeJS, Angular, VueJS e, claro, React. Elas conseguem expandir muito os casos de uso, permitindo que um número cada vez maior de programadores migrem para o JavaScript, aumentando o mercado de trabalho. Além disso, a comunidade JavaScript (e React) é enorme, ajudando muito no momento de buscar soluções e tirar dúvidas online.
